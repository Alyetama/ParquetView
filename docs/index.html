ParquetView

A fast, native macOS viewer for Apache Parquet files.

Download for macOS Free · Apple Silicon & Intel · unsigned (see how to open below)

What it does

Everything you need to browse Parquet, nothing you don't.

Handles huge files

Reads only the row groups on screen — multi‑gigabyte files open instantly.

Sort & filter

Sort any column; quick search or multi‑condition advanced filters (regex, ranges, AND/OR).

Full metadata

Schema, types, file size, compression codec, row‑group count, writer, format version.

Open any way

File picker, drag‑and‑drop, or Finder's “Open With”.

Inspect cells

Double‑click a cell to copy its value or edit it inline.

Light & dark

Theme, row density, and font size — remembered between launches.

How to open the app

ParquetView isn't signed with an Apple Developer ID, so macOS blocks it the first time. This is expected — open it with one of these:

  1. Right‑click to open. In Finder, right‑click (or Control‑click) ParquetView → Open → Open.
  2. If that's blocked on newer macOS: open System Settings → Privacy & Security, scroll down, and click Open Anyway next to the ParquetView message.
  3. Terminal fallback: remove the quarantine flag, then open normally: 
    /usr/bin/xattr -dr com.apple.quarantine /Applications/ParquetView.app
